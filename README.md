# README 파일이란?

이곳은 *저장소에 대한 설명*을 작성합니다.

**마크다운 문법**을 사용합니다.

## 깃허브 사용법

1. 원격 저장소 만들기
2. 지역 저장소 연결하기
   - origin 연결하기 
3. push, pull 하기
   - 원격 저장소에 파일 올리기
   - 원격 저장소에서 직접 커밋하기
   - 원격 저장소에서 커밋 내려받기
4. 협업하기
   1. 협업을 위한 저장소를 만든다.
   2. 팀원을 추가한다.
   3. 브랜치별~전체로~로 커밋하고 푸시한다.
   4. 풀 리퀘스트를 요청한다.
   5. 코드 리뷰 & 브랜치 병합

## 깃허브로 할 수 있는 일

- ***원격 저장소***에서 **깃**을 사용할 수 있다.
- ***지역 저장소***를 **백업**할 수 있다.
- 온라인 개발 툴을 사용할 수 있습니다.
- **협업 프로젝트**에 사용할 수 있다.
